OutputStyle

AbstractYes
OutputStyle.php : 31

Abstract base helper for extending Symfony Console output styles.

Provides a unified interface around OutputInterface, adding convenience methods to access, configure, and control console output behaviors.

Custom styles such as JsonStyle can extend this class to inherit all base features while adding specialized formatting logic.

Example:

use Symfony\Component\Console\Output\ConsoleOutput;
use oihana\commands\styles\JsonStyle;

$output = new ConsoleOutput();
$style  = new JsonStyle($output);
$style->writeln('<info>Hello world!</info>');
